Sobre HTML & CSS

HTML (HyperText Markup Language) é a estrutura da página. CSS (Cascading Style Sheets) é o visual da página.

💡 Dica: Clique com o botão direito na página e escolha "Inspecionar" para ver o código funcionando ao vivo no navegador!
🏗️

HTML

Estrutura e conteúdo da página. Define títulos, parágrafos, imagens e links.

🎨

CSS

Visual e aparência. Define cores, tamanhos, fontes e posicionamento.

⚡

JavaScript

Comportamento e interatividade. Faz a página reagir a ações do usuário.

Tags Essenciais do HTML

Tag O que faz Exemplo
<h1> ~ <h6> Títulos <h1>Título Principal</h1>
<p> Parágrafo <p>Meu texto aqui.</p>
<a> Link <a href="url">Clique aqui</a>
<img> Imagem <img src="foto.jpg" alt="descrição">
<ul> / <ol> Lista <ul><li>Item</li></ul>
<div> Caixa (bloco) <div class="minha-classe">...</div>
<span> Caixa (inline) <span style="color:red">texto</span>

Imagens no HTML

A tag <img> insere imagens. Veja os atributos importantes:

src = caminho da imagem  |  alt = texto alternativo (acessibilidade)  |  width / height = tamanho em pixels
